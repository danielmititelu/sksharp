# SkSharp

An unofficial library for Skype Api inspired by [SkPy](https://github.com/Terrance/SkPy)

### Skype Http API using REST Client 
```
    "rest-client.environmentVariables": {
        "$shared": {
            "username": "",
            "password": "",
            "chatName": "",
            "chatId": "",
            "userId": "",
            "securityToken": "",
            "registrationToken": ""
        }
    }
```
